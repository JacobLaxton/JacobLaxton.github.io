This is the code for the circuit, which has two parts. The first part sends the data from the Analog In pins (which are connected to the Joystick) over the serial port to be interpreted by the P5 javascript code below. The second part recieves the data from the P5 javascript, which is sending the keypresses from the keyboard. Depending on if a keypress is w or s, the voltage to the LED will increase or decrease.




This is the P5 javascript code that is run on the 'Web Server for Chrome' program. This code sets up the conenction to the serial port and creates a canvas for the program to draw on. The SerialEvent function reads the joystick data over the serial port, and the keyPressed fucntion reads the input form my keyboard. In the draw function, the characters and brightnes are being updated over every loop.
